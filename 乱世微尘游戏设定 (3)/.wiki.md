# Project Summary
"Wandering Microbe" is a narrative-driven role-playing game that immerses players in a rich historical environment, focusing on resource management, relationship building, and survival challenges. The game emphasizes interactive storytelling, enabling players to shape their unique journeys while exploring a mysterious world. Recent updates enhance the gameplay experience by allowing dynamic character attributes and improving game session management.

# Project Module Description
- **Backend**: Manages game logic, player data, and database interactions, including a random generation system for character attributes and events. Recently updated to include new columns for resources and companions in game sessions.
- **Frontend**: Provides the user interface for character creation, game world interaction, and gameplay management. Recent UI adjustments enhance the mystery and exploration feel.

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.

2. **Build the Project**:
   - For the backend: Use `alembic` for migrations.
   - For the frontend: Run `pnpm run build`.

3. **Run the Application**:
   - Start the backend server with `uvicorn main:app --reload` (from `app/backend`).
   - Start the frontend application with `pnpm run dev` (from `app/frontend`).
